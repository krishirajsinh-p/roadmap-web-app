## step 1: clone shallow copy
```bash
git clone --depth=1 git@github.com:krishirajsinh-p/roadmap-web-app.git
```
## step 2: install dependencies
```bash
npm install
```
## step 3: run
```bash
npm start
```
## step 4: open
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
